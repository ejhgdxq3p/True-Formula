# 真成分 True Formula 易拉宝生成器

本项目包含一个专门用于生成"真成分"易拉宝（Rollup Banner）设计的工具。

## 📋 规格参数

- **尺寸**：80cm × 200cm
- **分辨率**：150 DPI
- **像素尺寸**：4724px × 11811px
- **色彩模式**：导出为 RGB（打印建议转换为 CMYK）
- **风格**：复古游戏机 / 博物馆展板风格

## 🚀 使用方法

1.  **打开生成器**
    在浏览器中直接打开 `public/rollup-banner.html` 文件。
    
    > 建议使用 Chrome 浏览器以获得最佳渲染效果。

2.  **预览**
    页面中心显示的是缩小的预览图（约为实际尺寸的 12%），用于检查排版和内容。

3.  **导出文件**
    页面右上角有两个控制按钮：
    
    - **下载 PDF (80×200cm)**：生成符合打印规格的高清 PDF 文件。
        * *注意*：由于文件尺寸极大，生成过程可能需要 1-2 分钟，请耐心等待。
    - **下载 PNG 预览**：生成全尺寸的 PNG 图片。

## ⚠️ 打印注意事项

由于浏览器生成 PDF 的限制，导出的文件默认为 **RGB** 色彩模式。为了获得最佳打印效果（特别是在易拉宝材质上），建议：

1.  下载生成的 PDF 文件。
2.  使用 Adobe Acrobat Pro 或 Photoshop 打开。
3.  将色彩模式转换为 **CMYK**。
4.  检查黑色文字是否为单色黑（K100），或者保留设计中的深绿色（#0F380F）。

## 🛠️ 技术实现

- **HTML5/CSS3**：用于精确的像素级排版。
- **html2canvas**：将 DOM 节点渲染为 Canvas。
- **jsPDF**：将 Canvas 封装为标准 PDF 文档。
- **Google Fonts**：使用 'JetBrains Mono' 和 'Noto Sans SC' 字体。

## 📝 内容维护

如果需要修改文案，请直接编辑 `public/rollup-banner.html` 文件中的 HTML 文本内容。所有样式类（如 `.section-title`, `.paragraph-cn`）已封装好，修改文本不会影响整体布局。
